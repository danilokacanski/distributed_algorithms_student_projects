# IBFT Consensus Simulator

Ovaj projekat je simulacija Istanbul Byzantine Fault Tolerance (IBFT) algoritma konsenzusa u jeziku Go.

## Kako pokrenuti projekat
1. Instalirajte [Go](https://go.dev/dl/) (verzija 1.22 ili novija).
2. Otvorite terminal u osnovnom folderu projekta.
3. Pokrenite komandu: `go run main.go`

## Dostupni scenariji
U glavnom meniju možete izabrati jedan od 7 scenarija, uključujući:
- Normalno izvršavanje
- Otkaz lidera
- Vizantijsko ponašanje
- Test bezbednosti (Justify mehanizam)
