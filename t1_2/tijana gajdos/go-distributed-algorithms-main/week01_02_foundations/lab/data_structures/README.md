# Data Structures

This module explores Go's built-in composite data types for organizing and storing data.

## Topics Covered

### 10_arrays
- Fixed-size arrays
- Array declaration and initialization
- Array indexing and bounds
- Array iteration with `for` loops

### 11_slices
- Dynamic arrays (slices)
- Slice declaration and initialization
- Slice operations (append, copy, etc.)
- Slice capacity and length
- Slice expressions and reslicing

### 12_slicing
- Advanced slicing techniques
- Slice manipulation
- Memory sharing between slices
- Slice bounds and capacity

### 13_strings2
- Advanced string operations
- String manipulation functions
- String conversion
- String searching and replacing

### 14_maps
- Key-value data structures
- Map declaration and initialization
- Map operations (insert, delete, lookup)
- Map iteration
- Map zero value and nil maps

### 15_structures
- Custom data types
- Struct declaration and fields
- Struct initialization
- Anonymous structs
- Struct embedding and composition
