# Language Basics

This module covers the fundamental building blocks of Go programming language.

## Topics Covered

### 01_hello_world
- Introduction to Go syntax
- Basic program structure
- `package main` and `func main()`

### 02_primitive_data_types
- Boolean types (`bool`)
- Numeric types (`int`, `float64`, etc.)
- String types (`string`)
- Basic type declarations and usage

### 03_integer_data_types
- Signed and unsigned integers (`int8`, `uint16`, etc.)
- Integer overflow and underflow
- Type conversions between integers

### 04_floats_and_complex_numbers
- Floating-point types (`float32`, `float64`)
- Complex number types (`complex64`, `complex128`)
- Precision and rounding issues

### 05_strings
- String literals and declarations
- String concatenation
- String indexing and slicing
- Unicode and UTF-8 support

### 06_unpacking_variables
- Multiple assignment
- Variable swapping
- Ignoring values with `_`
- Tuple unpacking

### 07_constants
- `const` keyword
- Typed and untyped constants
- Constant expressions
- `iota` enumerator

### 08_identifiers_and_keywords
- Valid identifier naming rules
- Reserved keywords
- Exported vs unexported identifiers
- Naming conventions

### 09_operators
- Arithmetic operators
- Comparison operators
- Logical operators
- Bitwise operators
- Operator precedence
